javascript-countdown
====================

A Simple JavaScript Count Down

Usage
==========

Just set below variables in source code <code>countdown.html</code> to your target date.
<pre>
    var y = 2014; // Year
    var M = 03; // Month
    var d = 17; // Day
    var h = 19; // Hour
    var m = 30; // Minute
</pre>
